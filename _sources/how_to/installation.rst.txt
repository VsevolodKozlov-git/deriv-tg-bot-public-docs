Установка
=========


Подключение бота и чата в телеграмм
-----------------------------------
#. Создайте бота в телеграмм. `Гайд по созданию <https://sendpulse.com/ru/knowledge-base/chatbot/telegram/create-telegram-chatbot>`__
#. Создайте публичную группу в телеграмм
#. Добавьте созданного бота в группу
#. В модуле ``deriv-telegram-bot/app/config.py``:
    #. В поле ``telegram_token`` впишите токен от бота
    #. В поле ``telegram_group`` впишите тег группы в формате ``@your_group_name``


Запуск приложения
-----------------

Вариант 1. Запуск из исходников(рекомендуется)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
#. Установить python версии 3.9.6

    .. danger:: На других версиях Python бот может не запуститься.

#. Установите pip
#. Перейдите в директорию deriv-telegram-bot, то есть в корневую директорию
#. В терминале напишите ``pip install -r requirements.txt``, чтобы установить все зависимости
#. Добавить директорию deriv-telegram-bot в PYTHONPATH
#. Напишите  в терминале ``python app/main.py``, чтобы запустить бота

Вариант 2. Установить приложение как пакет
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

#. Установить python версии 3.9.6

    .. danger:: На других версиях Python бот может не запуститься.

#. Установите pip
#. Перейдите в директорию deriv-telegram-bot, то есть в корневую директорию проекта
#. В терминале напишите ``pip install -e .``, чтобы установить проект, как пакет

    .. note:: ключ ``-e`` используется для того, чтобы изменения в коде сразу же вступали в силу. В противном случае придется после каждого изменения кода придется переустанавливать пакет.

#. Чтобы запустить бота напишите в терминале ``run_deriv``
