sender package
==============

Пакет отвечает за:

* Получение уведомлений типа :py:class:`app.sender.notification.CpNotification` от :py:mod:`app.conditions.conditions`.
* Форматирование уведомлений в удобочитаемую строку
* Асинхронную отправку сообщений
* Соединение с api telegram
