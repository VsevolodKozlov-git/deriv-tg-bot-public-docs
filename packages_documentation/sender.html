sender package#

Пакет отвечает за:

  • Получение уведомлений типа app.sender.notification.CpNotification от app.conditions.conditions.

  • Форматирование уведомлений в удобочитаемую строку

  • Асинхронную отправку сообщений

  • Соединение с api telegram

async notification_sender_task()#

Асинхронная бесконечная функция для отправки уведомлений в MsgSender

  1. Ждет поступления уведомления

  2. Засыпает на app.config.Config.notification_queue_sleep, чтобы дождаться прихода всех уведомлений

  3. Форматирует уведомления в строку

  4. Отправляет их в MsgSender
