  • Прерывают и создают новые подписки

class app.subscription.subsription_classes.SubscriptionObserver(condition_partial: partial[ConditionABC], currency_pair: str, data_checker: SubscriptionCheckerOnePair, granularity: int, log=False)#

Обрабатывает свечи, получаемые по подписке. Для каждой подписки должен создаваться свой SubscriptionObserver

Parameters:

  • condition_partial – экземпляр functools.partial с экземпляром наследника внутри ConditionABC внутри. То есть при вызове condition_partial() вернется готовый к использованию экземпляр наследника ConditionABC. Это сделано для того, чтобы каждый SubscriptionObserver создавал независимый экземпляр условия

  • currency_pair – валютная пара

  • data_checker – экземпляр класса SubscriptionCheckerOnePair

  • granularity – интервал свечи

  • log – сохранять ли информацию, о полученных свечах в логи

class app.subscription.subsription_classes.SubscriptionsGenerator(cond_pair_granularity: List[Tuple[partial[ConditionABC], str, int]], log: bool)#

Функции класса:

  • Генерация подписок

  • Создание SubscriptionObserver для них

  • Заполнение SubscriptionCheckersManager экземплярами SubscriptionCheckerOnePair

Parameters:

  • cond_pair_granularity – Список кортежей вида (условие, валютная пара, промежуток свечи)

  • log – Сохранять ли информацию о полученных свечах в логи? Далее этот параметер передается в SubscriptionObserver
