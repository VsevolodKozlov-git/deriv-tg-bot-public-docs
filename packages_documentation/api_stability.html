api_stability package#

Модули, отвечающие за стабильность работы api.

internet module#

Проверка соединения с интернетом

class app.api_stability.internet.InternetReconnecter#

Выполняет проверку соединения с интернетом и переподключение при необходимости

static _checker_delay_iter()#
Returns:

Промежутки перед повторным подключением. Промежутки берутся из server_is_unreachable_sleep

static have_internet() → bool#
Returns:

Есть ли соединение с интернетом?

async reconnect_if_needed()#

Проверяет возможность повторного подключения через промежутки времени из _checker_delay_iter()

subscription module#

Модуль содержит 2 класса для проверки получения информации по подписке

class app.api_stability.subscription.SubscriptionCheckerOnePair#

Проверяет одну пару на получение данных

seconds_passed()#
Returns:

Количество секунд с последнего обновления

update_last_date()#

Обновляет последнее время получения данных

class app.api_stability.subscription.SubscriptionCheckersManager#

Агрегирует в себе SubscriptionCheckerOnePair. Через промежутки времени проверяет, все ли пары получают данные.

add__data_checker(data_checker: SubscriptionCheckerOnePair)#

Добавить экземпляр SubscriptionCheckerOnePair

async checker_coroutine()#

Асинхронная функция, которая возвращает ошибку, если хотя бы один SubscriptionCheckerOnePair не получает данные дольше, чем max_data_gap

Raises:

ApiDataNotArriveError – данные не получаются слишком долго

delete_data_checkers()#

Удалить все экземпляры SubscriptionCheckerOnePair

is_everything_updates()#
Returns:

Все ли экземпляры SubscriptionCheckerOnePair были обновлены раньше, чем прошло max_data_gap

update_all_data_checkers()#

Обновить все экземпляры SubscriptionCheckerOnePair
